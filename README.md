# Portafolio Personal Web

## Descripción
Proyecto de sitio web personal desarrollado como evaluación final del curso de Desarrollo Web.
Incluye una página principal con información biográfica y un formulario de contacto funcional 
que guarda los mensajes en una base de datos MySQL.

## Tecnologías utilizadas
- HTML5 (Semántico)
- CSS3 (Diseño responsivo)
- PHP (Lógica del servidor)
- MySQL (Base de datos)

## Estructura de archivos
- `index.php`: Página de inicio (Biografía).
- `contact.php`: Formulario de contacto y lógica PHP.
- `style.css`: Estilos visuales.
- `db.php`: Archivo de conexión a la base de datos.
- `README.md`: Este archivo.

## Instrucciones de uso local
1. Instalar XAMPP o WAMP.
2. Iniciar Apache y MySQL en el panel de control.
3. Crear una base de datos en phpMyAdmin con el nombre definido en db.php.
4. Importar la tabla (usar el script SQL proporcionado).
5. Ejecutar el archivo index.php desde el navegador (localhost).
